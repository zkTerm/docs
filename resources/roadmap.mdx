# Roadmap

## Q4 2025

### Phase 1

1. zkID Authentication - `did:zk:xxx` identity with Ed25519 proofs
2. Solana Wallet - BIP39 + AES-256-GCM encryption
3. XTerm.js Terminal - CLI with SSE streaming

### Phase 2

1. ZK Compression - Light Protocol integration
2. Starknet Wallet - OpenZeppelin/ArgentX compatible
3. zkStorage - IPFS via Pinata with Groth16 proofs
4. zkAttest - Privacy-preserving data attestation
5. zkToolkit - 10 cryptographic modules
6. Trade-to-Earn Rewards

### Phase 3

1. zkAuth - Passwordless authentication with STARK proofs
2. zkExplorer - Visual dashboard for wallets, files, and transactions
3. X402 Integration - Internal payment system
4. Zcash Shielded Address - Z-address support
5. ZK Mixer with Zcash - Private transfers via shielded hop
