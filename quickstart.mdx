## 5-Minute Quickstart

This guide will get you up and running with zkTerm in just 5 minutes.

## Step 1: Access Terminal

Open the zkTerm terminal:

```
https://zkterm.io/terminal
```

## Step 2: Create Identity

Run your first command:

```bash
zkid create
```

Enter a username and password when prompted. You'll receive:
- Your unique `did:zk` identifier
- Ed25519 public key
- On-chain verification on Solana + Starknet

## Step 3: Try Cryptographic Primitives

Hash some data:

```bash
zk toolkit hash poseidon "hello zkTerm"
```

Do field arithmetic:

```bash
zk field add 42 58
```

Generate a proof:

```bash
zk proof generate --inputs "a=5,b=7"
```

## Step 4: Upload Encrypted File

Create a test file and upload:

```bash
echo "Secret data" > test.txt
zk storage upload test.txt
```

You'll get an IPFS CID like `QmXyz123...`

Download it back:

```bash
zk storage download QmXyz123...
```

## Step 5: Compress Tokens (Optional)

If you have SOL or SPL tokens:

```bash
# View your tokens
zk tokens

# Compress USDC (if you have it)
compress EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v 10

# Check compressed balance
zk tokens
```

## Common Commands

| Command | Description |
|---------|-------------|
| `zkid create` | Create privacy-preserving identity |
| `zkid verify` | Verify identity on-chain |
| `zk toolkit help` | List all cryptographic modules |
| `zk storage upload <file>` | Encrypt and upload to IPFS |
| `compress <mint> <amount>` | Compress Solana tokens |
| `zk tokens` | View token portfolio |
