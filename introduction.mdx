## What is zkTerm?

zkTerm solves the biggest problem in zero-knowledge development: **ecosystem fragmentation**. Instead of learning different tools for each ZK technology, zkTerm provides one unified interface for:

- **zkID** - Privacy-preserving universal identity
- **zkCompression** - Solana state compression (1000x cost reduction)
- **zkToolkit** - Complete cryptographic primitives library
- **zkStorage** - Encrypted IPFS with zkSNARK proofs
- **zkAuth** - Passwordless social login with STARK proofs

## Quick Start

### 1. Access the Terminal or Explorer

Visit [zkterm.io/terminal](https://zkterm.io/terminal) for the CLI or [zkterm.io/explorer](https://zkterm.io/explorer) to browse zkTerm user data.

REST API available at `https://zkterm.io/api/`

### 2. Create Your Identity

```bash
zkid create
```

Generate a privacy-preserving `did:zk` identity with dual-chain verification (Solana + Starknet).

### 3. Explore Cryptographic Primitives

```bash
zk toolkit help
```

Access 10 modules: hash, commit, merkle, range, sign, nullifier, field, ec, shamir, proof.

### 4. Upload Encrypted Files

```bash
zk storage upload secret.pdf
```

Client-side AES-256-GCM encryption with Groth16 zkSNARK password verification.

### 5. Compress Solana Tokens

```bash
compress <mint_address> <amount>
```

Reduce storage costs by 1000x using Light Protocol compression.

## Architecture

zkTerm is built on a modern, scalable stack:

```
┌─────────────────────────────────────────┐
│         zkTerm Terminal (CLI)           │
│   React + TypeScript + Vite + XTerm     │
└─────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────┐
│         REST API (Express.js)           │
│    zkID • zkStorage • zkToolkit         │
│    zkCompression • zkAuth               │
└─────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────┐
│      Blockchain & Storage Layer         │
│  Solana • Starknet • IPFS • PostgreSQL  │
└─────────────────────────────────────────┘
```

**Tech Stack:**
- **Frontend**: React, TypeScript, Vite, TailwindCSS
- **Backend**: Express.js, Node.js
- **Database**: PostgreSQL with Drizzle ORM
- **Blockchain**: Solana (Helius RPC), Starknet
- **ZK Libraries**: circomlibjs, snarkjs, starknet.js
- **Storage**: IPFS (Pinata), Google Cloud Storage

## Key Features

### zkID - Universal Identity

```bash
zkid create          # Create privacy-preserving identity
zkid verify          # Generate Ed25519 proof
zkid status          # Check on-chain verification
```

- `did:zk:[12 chars]` universal DID format
- Dual-chain verification (Solana + Starknet)
- Ed25519 cryptography
- Cairo smart contract on Starknet

### zkCompression - Solana Optimization

```bash
compress <mint> <amount>     # Compress tokens
decompress <mint> <amount>   # Decompress tokens
zk tokens                    # View portfolio
```

- 1000x storage cost reduction
- Light Protocol integration
- Automatic retry polling
- Full SPL token compatibility

### zkToolkit - Cryptographic Primitives

```bash
zk toolkit hash poseidon "data"    # Hash functions
zk field add 5 7                   # Field arithmetic
zk ec mulbase 12345                # Elliptic curves
zk proof generate                  # Groth16 proofs
```

- 10 complete modules
- ZK-circuit optimized
- Production-ready primitives
- Command-line + API access

### zkStorage - Encrypted Files

```bash
zk storage upload <file>       # Encrypt & upload
zk storage download <cid>      # Download & decrypt
zk storage verify <cid>        # Verify ZK proof
```

- Client-side AES-256-GCM encryption
- IPFS storage via Pinata
- Groth16 zkSNARK proofs
- SHA-256 checksum validation

### zkAuth - Passwordless Login

```bash
zkauth login google           # Login with Google
zkauth login github           # Login with GitHub
zkauth status                 # Check auth status
```

- Web3Auth social login
- STARK proofs on Starknet
- No passwords or seed phrases
- Quantum-resistant cryptography

### zkTerm Explorer - Decentralized Data Browser

**Visit:** [zkterm.io/explorer](https://zkterm.io/explorer)

- Browse all zkTerm users and their zkIDs
- View Starknet & Solana wallets with real-time balances
- Explore transaction history (compressions, uploads, verifications)
- Live activity feed across the network
- **Fully decentralized:** Hosted on IPFS via Crust Network
- **No servers, no downtime** - unstoppable interface
- Privacy-preserving: Only public blockchain data visible
